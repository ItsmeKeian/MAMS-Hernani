# Municipal Aid Monitoring System

A web-based **Municipal Aid Monitoring System** developed for the Municipality of Hernani to manage, track, and monitor the distribution of financial assistance, relief goods, and other social welfare support to beneficiaries.  
This system was created to replace manual paper-based recording and spreadsheet tracking with a centralized digital platform.

---

## 📌 Project Description

Local Government Units (LGUs) play a vital role in delivering social welfare services, especially during disasters, emergencies, and poverty-related assistance programs.  
Manual recording of beneficiaries often leads to data duplication, slow verification, and difficulty in generating reports.

This project provides a **centralized web-based system** that allows authorized personnel to:

- Record beneficiary information
- Track aid distribution
- Prevent duplicate records
- Generate reports
- Improve transparency and accuracy

The system is designed for **local deployment (localhost / intranet)** and intended for use by municipal staff only.

---

## 🎯 Objectives

### General Objective
To develop a centralized web-based Municipal Aid Monitoring System that improves the efficiency, transparency, and accuracy of aid distribution.

### Specific Objectives

- Create a centralized database for beneficiary records
- Track financial aid and relief distribution
- Prevent duplicate beneficiary entries
- Generate reports by date, barangay, and aid type
- Provide a user-friendly interface
- Improve data accuracy and monitoring

---

## 📊 Scope and Limitations

### Scope
- Beneficiary registration
- Family member records
- Assistance distribution tracking
- Report generation
- User login system
- Admin dashboard
- Web-based system (localhost)

### Limitations
- No mobile application
- No biometric verification
- No GPS tracking
- No integration with national databases
- Localhost deployment only
